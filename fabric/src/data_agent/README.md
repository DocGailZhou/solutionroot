# Data Agent Instructions 



Normally you only need to set up one data agent. You can choose one among below methods



(1) Use Ontology with data source as semantic model. And create data agent, with data source as this ontology. 

(2) Use Ontology Graph Model to link  data lake schema and tables, and enhance it. Then use this ontology as the data source for the data agent. 

(3) If you choose not using ontology, you can create data agent using lakehouse as the data source.
